Install-GpGarlic

ðŸ§„ Install GarlicOS!

Automate installation of GarlicOS.

Syntax 

Install-GpGarlic -LocalFile <String> [-TempPath <String>] -TargetDeviceNumber <Int32> [-ClearTempPath <Boolean>] [-BIOSPath <String>] [-ROMPath <String>] [-ExpandPartitionThresholdMb <String>] [-ROMDriveLetter <String>] [-WhatIf] [-Confirm] [<CommonParameters>]

Install-GpGarlic -GarlicURL <String> [-GarlicInstallZipName <String>] [-TempPath <String>] -TargetDeviceNumber <Int32> [-ClearTempPath <Boolean>] [-BIOSPath <String>] [-ROMPath <String>] [-ExpandPartitionThresholdMb <String>] [-ROMDriveLetter <String>] [-WhatIf] [-Confirm] [<CommonParameters>]

Parameters

Name Alias Description Required? Pipeline Input Default Value
-LocalFile The full path of a copy of RG35XX-MicroSDCardImage.7z locally available. If not supplied, it is downloaded. true false
-GarlicURL The URL of the GarlicOS update file (RG35XX-MicroSDCardImage.7z), ex: https://www.patreon.com/file?h=76561333&i=13249827
Note: With each new version of GarlicOS, old URLs become invalid. Ensure a valid one is being passed.		 true false
-GarlicInstallZipName When using GarlicURL, the name to save the downloaded file as in the TempPath. false false RG35XX-MicroSDCardImage.7z
-TempPath Where files will be downloaded and decompressed to during the installation. false false (Join-Path -Path ([System.IO.Path]::GetTempPath()) "\GarlicPs")
-TargetDeviceNumber The index of the target SD card device. Can be found using diskpart.exe or equivalent. true false 0
-ClearTempPath Whether to recursively empty the TempPath before using it. Recommended. false false True
-BIOSPath Path to personal BIOS files that will be copied after installation. false false
-ROMPath Path to personal ROM files that will be copied after installation. false false
-ExpandPartitionThresholdMb The threshold in MB of unallocated space on the target disk.
If exceeded, the ROM partition will be expanded to utilize the space.		 false false 64MB
-ROMDriveLetter If the ROM partition does not get assigned a drive letter on re-insert, assign it this one
to make it accessible.		 false false R
-WhatIf wi false false
-Confirm cf false false

Examples

EXAMPLE 1 
Install-GpGarlic -GarlicUrl "https://www.patreon.com/file?h=76561333&i=13249827" -TargetDeviceNumber 2 -ClearTempPath $true
Fetches GarlicOS from a Patreon attachment URL and installs it on an SD card identified as Disk #2. Clears any files that may exist in the temp path.
EXAMPLE 2 
Install-GpGarlic -LocalFile "C:\Users\lowlydba\Downloads\RG35XX-MicroSDCardImage.7z" -TargetDeviceNumber 1 -ClearTempPath $true -TempPath "C:\temp"
Uses a local GarlicOS file and installs it on an SD card identified as Disk #1. Stores temp files in C:\temp. Clears any files that may already exist in C:\temp. Requires -RunAsAdministrator
